## Agentic AI Harness — What It Is & Why It Matters

An agentic AI harness is the thin layer of infrastructure that sits between a raw LLM and a working AI application. It handles the ReAct loop (Reason + Act), tool dispatch, context management, and result aggregation — letting you focus on what the agent should do rather than how it executes.

Key angles to cover:
- What problem the harness solves vs. calling the LLM directly
- The five building blocks: system prompt, tools, memory, loop, output
- A minimal working example
- When you need a harness vs. when a single LLM call is enough
